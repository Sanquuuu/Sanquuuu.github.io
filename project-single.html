App Santé Connectée

Application web pour suivi des patients en temps réel – Full-stack Python + React

Contexte

Ce projet a été réalisé dans le cadre d’un atelier interdisciplinaire autour de la e-santé. L’objectif : développer une interface intuitive pour les médecins et les patients afin de visualiser des données de santé en direct.

Fonctionnalités

  • Dashboard en React
  • API sécurisée (Flask)
  • Base de données PostgreSQL
  • Authentification + responsive design
